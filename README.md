# Privacy Policy – Personal Portfolio

Last updated: March 2026

Personal Portfolio does not collect, store, or share personal data.

## Data Usage
- The app may access public market data APIs if the user provides their own API key.
- API keys are stored locally on the user’s device using Apple Keychain.
- No data is transmitted to any developer-controlled servers.

## Analytics & Tracking
- This app does not use analytics.
- This app does not track users.
